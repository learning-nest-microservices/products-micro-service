# Products Microservice

## Description

Microservice for managing products.

## Getting started

### Prerequisites

- Node.js
- Prisma

### Installation

```bash
$ npm install
```

### Database

Create a new database and run migrations:

```bash
$ npx prisma migrate dev
```

### Running the application

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
